# README: ScriptFlow

**Transform Your Google Apps Script Experience**

ScriptFlow is a powerful Chrome Extension designed for developers who want a more professional, productive, and beautiful coding environment within the Google Apps Script (GAS) editor.

## üõ† Installation

1. Download/Clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the extension folder.
